# Binaries

Various binaries to

* Clean (`bin/clean`)
* Compile (`bin/compile`)
* Autocompile (`bin/autocompile`)

the Thesis. Uses

* LatexMk
* Biber
* XeTeX (`-xelatex`)

## Installation

### Debian / Ubuntu

Use the provided file `install-dependencies` like this:

    bash bin/install-dependencies
